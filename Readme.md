Power BI Dashboard

Project Title:
Supply Chain & Sales Analytics Dashboard

Overview:
This Power BI dashboard provides insights into supply chain performance, supplier analysis, cost of goods sold (COGS), regional sales performance, and demand monitoring. The dashboard is designed to help stakeholders make data-driven decisions by tracking key business metrics across multiple dimensions.

Dashboard Pages
1. Overview

Purpose: High-level summary of business performance.

Visuals Used:
KPI Card
Line Chart
Slicer

Key Insights:
Overall business performance
Trend analysis over time
Interactive filtering for deeper analysis.

2. Supplier Details

Purpose: Analyze supplier contributions and performance.

Visual Used:
Decomposition Tree

Key Insights:
Supplier-wise breakdown
Performance drivers
Root-cause analysis of supplier metrics.

3. COGS (Cost of Goods Sold)

Purpose: Monitor and analyze product costs.

Visuals Used:
Table
Slicer
Treemap

Key Insights:
Cost distribution by category
Major cost contributors
Product-level cost analysis.

4. Sales by Region

Purpose: Evaluate sales performance across different regions.

Visuals Used:

Key Insights:
Regional sales comparison
Market contribution analysis
Top-performing regions.

Purpose: Track demand trends and performance indicators.
